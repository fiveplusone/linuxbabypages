linux baby pages

What

linux baby pages (lbp) is an effort to make concise “common uses” man pages.

Rules

  1. lbp entries can be no more than 2000 characters.
  2. The lbp corpus can be no more than 100,000 characters total.
  3. Spaces count, but not line breaks.
  4. lbp entries hard wrap at 72 characters.

Process

To suggest a change to the script or corpus, make a pull request to the appropriate repository.

Entries

.htaccess chmod chown cron curl du emacs find free git grep iptables kill ls mysql mysqldump ps rsync sendmail sudo tar vim wc wget which

Related

There are at least two projects with similar goals that predate lbp: tldr and bro.
